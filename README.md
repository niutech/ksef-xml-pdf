# Wizualizator e-Faktur KSeF z XML do PDF

Wygeneruj wizualizację e-faktury z KSeF z pliku XML do pliku PDF.

Narzędzie działa po stronie klienta, nie przechowuje żadnych danych na serwerach, korzysta z biblioteki [ksef-pdf-generator](https://github.com/CIRFMF/ksef-pdf-generator).

#### [Przetestuj online](https://ai-gov.pl/ksef-pdf/)
